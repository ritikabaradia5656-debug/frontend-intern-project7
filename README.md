## 🚀 Features

- 🔍 Search movies (OMDB API)
- 🎬 Netflix-style UI
- 📄 Pagination
- ❤️ Add/Remove Favorites (localStorage)
- 📱 Fully responsive

---

## 🛠 Tech Stack

- React (Vite)
- JavaScript
- CSS (Netflix UI)
- OMDB API
